# streak-keeper

> [!IMPORTANT]
> don't expect this to _ever_ be fully functioning. i'm making this just as a fun project in my free-time as a means of getting better at rust and web development[^1]. if/when i ever deploy this, i will hotlink the website here.

### description
this is a project dedicated to keeping track of dead by daylight streaks, and additionally providing a simple, pretty interface for streak overlays to display via a browser source on OBS.

as of the time of writing this, it is intended that the website is built with axum for the backend, and sveltekit for the frontend.

### features
- [ ] supports discord oauth for logins
- [ ] websocket for real-time updates
- [ ] add social aspects? (private/public profiles, the ability to view people's streaks, etc.)
